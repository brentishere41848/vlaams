Multi-Vlaams Dialects

Multi-Vlaams dialecten

Vlaams Codex supports multiple Flemish dialect command aliases. Pick the dialect that feels natural to you and keep coding the same way.

Command aliases

Commando-aliassen

The CLI is called plats. The first word after plats can be written in different dialects.

De CLI heet plats. Het eerste woord na plats kunt ge schrijven in verschillende dialecten.

Notes

Opmerkingen

Dialect aliases only change the command words. Your .plats code stays the same.

Dialect-aliassen veranderen enkel de commando-woorden. Uw .plats-code blijft hetzelfde.

If you prefer, you can always stick to Standard. Nothing is required.

Als ge wilt, kunt ge altijd Standard blijven gebruiken. Niks is verplicht.
