Quick Start

Get started with Vlaams Codex in seconds

1

Install

# Using pip
pip install vlaamscodex

# Or using npm
npm install vlaamscodex
2

Run Your First Program

plats run hello.plats
3

Or Use Magic Mode

python hello.plats

Installation Options

Recommended

pipx (Isolated)

pipx install vlaamscodex

Perfect for end users. Keeps dependencies isolated.

pip (Standard)

pip install vlaamscodex

Standard installation. Works with existing Python setup.

Development

git clone https://github.com/brentishere41848/Vlaams-Codex.git
cd Vlaams-Codex
pip install -e ".[dev]"

For contributors. Includes test suite and development tools.

NPM

npm

npm install vlaamscodex

Install for Node.js projects. Available now!
